#  Sudoku Solver in C++ (CodeAlpha Internship)

## Project Overview
This project is a **console-based Sudoku Solver** developed using **C++**.  
It uses the **Backtracking algorithm** to solve a 9×9 Sudoku puzzle by filling empty cells while satisfying Sudoku constraints.

---

## Features
- Solves standard 9×9 Sudoku puzzles
- Uses recursion and backtracking
- Validates row, column, and 3×3 subgrid rules
- User input based

---

## Technologies Used
- C++
- Recursion
- Backtracking Algorithm
- 2D Arrays

---

```bash
g++ sudoku_solver.cpp -o sudoku
